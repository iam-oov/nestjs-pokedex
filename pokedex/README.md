# Mode Development
1. Clonar el repositorio
2. Ejecutar
```
yarn install
```
3. Tener nest CLI instalado
```
npm i -g @nestjs/cli
```
4. Levantar la DB
```
docker-compose up -d
```

# Stack usado
* MongoDb
* Nest
